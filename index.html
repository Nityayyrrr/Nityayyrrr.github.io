MedRoute

Real-time emergency ambulance coordination platform connecting patients, drivers, and hospitals. Built with React Native + Expo for mobile clients and a Node.js + Socket.io backend powering live bidirectional communication. Implements a geospatial dispatch engine using MongoDB 2dsphere indexing combined with a weighted scoring algorithm across live bed availability and real-time ETA data. Redis manages ephemeral state including driver GPS tracking and hospital bed counts, enabling sub-second updates across multiple user roles.
